== Description ==

This plugin makes available a custom endpoint on the WordPress site
On visiting the endpoint a list of Users is retrieved from a remote API and some data about each is displayed
 in an interactive HTML table.
Additional data about each user can be  retrieved from the remote API on clicking that users row.
